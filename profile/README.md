Welcome to the official home of **Zentra Core**. We are developing a local-first, modular architecture designed to bring high-level autonomous intelligence to private environments.

### 🏛️ Our Mission
To create a seamless, private, and secure bridge between human intent and machine execution. Zentra Core is built to act as the central "brain" orchestrating diverse environments—from smart living spaces to autonomous software environments and advanced robotics.

---

### 🚀 Key Focus Areas
* **Local-First & Privacy by Design:** Everything runs securely on your hardware. Your data, your rules. No mandatory cloud dependency.
* **Modular Intelligence:** A dynamic "plug-and-play" plugin architecture where you can expand capabilities via Native Function Calling.
* **Autonomous Agentic Loop:** Zentra actively reasons, selects tools dynamically, and executes multi-step chains of thought.
* **Multimodal Interoperability:** Native vision support (LLaVA, Gemini, OpenAI), text-to-speech integration, and seamless API bridging.

### 🛠️ Project Status: Runtime Alpha
Zentra Core has reached **Version 0.9.9 (Runtime Alpha)**. 
We have successfully implemented the stable core architecture, featuring real-time streaming, a dedicated multi-language WebUI, and full multi-backend model support (Ollama, KoboldCpp, Groq, OpenAI, Anthropic, Gemini). Our current focus is expanding the cognitive memory layer and solidifying the autonomous agentic capabilities.
